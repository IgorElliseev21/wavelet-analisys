В ноуте Class_Signal.ipynb реализован класс сигнал с методами каскадного синтеза и анализа, а также функция факторизации для построения пары фильтров из многочлена с одной нечётной степенью.
Также представлены результаты применения каскадного синтеза к файлу test.wav и последующем синтезе с заменой одного из сигналов разложения нулевым.
Разложение выполнялось на глубину 4, цифра в конце названия файла обознаяает номер разложенного сигнала, заменённого нулём.
Заметно, что при занулении первого сигнала остаётся только небольшое количество верхних частот(степень изменения зависит от применяемого фильтра), зануление дальнейших сигналов даёт малозаментый на слух результат
